## 🎯 Quick Facts

**Market Opportunity:**
- Global HR Analytics Market: $9.2B (2024)
- CAGR: 18.2% annually
- Target TAM (mid-market): $63.75B
- Target SOM (Year 1-3): $150-200M

**Target Customer:**
- Size: 250-2,000 employees
- Geography: US, EU, APAC
- Budget: $20-75k annually
- Pain: Turnover >25%, data silos, compliance

**Key Differentiators:**
- ✅ Mobile-first design
- ✅ Real-time analytics
- ✅ AI-powered recommendations
- ✅ No-code for HR teams
- ✅ 3-5x affordable than competitors
- ✅ Days to implement vs months

**Competitive Positioning:**
- Incumbent: SAP, Workday, Oracle (expensive, legacy)
- Emerging: Visier, Lattice, Culture Amp (enterprise-focused)
- **Our Niche:** Modern, affordable, mobile-first for mid-market
