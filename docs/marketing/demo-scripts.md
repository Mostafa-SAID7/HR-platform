## Demo Best Practices

### Setup
- [ ] Test audio/video before starting
- [ ] Have live data (not fake demo data)
- [ ] Know your company's use case before demo
- [ ] Have ROI calculator prepared

### During Demo
- [ ] Ask permission before drilling into data
- [ ] Explain "why," not just "how"
- [ ] Use real company examples when possible
- [ ] Pause for questions every 2-3 minutes
- [ ] Focus on their #1 pain point first

### Closing
- [ ] Summarize benefits (what we showed them)
- [ ] Address concerns (objections)
- [ ] Clear next steps (trial, meeting, pricing)
- [ ] Send materials within 24 hours

**Q: "How long to see ROI?"**
