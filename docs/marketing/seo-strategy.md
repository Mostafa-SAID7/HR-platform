## On-Page SEO Checklist

### Every Blog Post Must Have:

**Title & Meta**
- [ ] Primary keyword in title (focus keyword)
- [ ] Title 50-60 characters
- [ ] Meta description 120-160 characters
- [ ] Meta includes CTA if applicable

**Content**
- [ ] H1 tag (only 1, includes primary keyword)
- [ ] H2 tags (2-4, includes related keywords)
- [ ] H3 tags (under each H2, supports content)
- [ ] 2,000+ words (for pillar content)
- [ ] 1,500+ words (for supporting content)
- [ ] Readability score 60+
- [ ] Active voice (70%+)

**Links**
- [ ] Internal links: 5-10 (to related content)
- [ ] External links: 2-3 (to authoritative sources)
- [ ] Anchor text natural (keyword-rich but conversational)
- [ ] Links open in new tab

**Media**
- [ ] Images: 3-5 throughout post
- [ ] Image alt text (keyword-rich, descriptive)
- [ ] Image file names descriptive
- [ ] Video if applicable (increases engagement)

**Technical**
- [ ] URL slug (short, keyword-rich, no dates)
- [ ] Schema markup (Article schema minimum)
- [ ] Mobile responsive (test on mobile)
- [ ] Load time < 3 seconds
- [ ] No broken links

## Link Building Strategy

### Internal Linking
- **Pillar pages:** Link to 10-15 supporting content pieces
- **Supporting content:** Link back to pillar
- **Related posts:** Link to 3-5 contextually related posts
- **Strategy:** Create internal link network for topic authority

### External Linking
- **Authority:** Link to 2-3 trusted sources per post
- **Diversity:** Mix of .edu, .gov, industry sites
- **Anchor text:** Natural, keyword-rich
- **Relevance:** Only link to truly related content

### Earning Backlinks

**Content-Based:**
1. Create original research (survey, data report)
2. Create comprehensive guides
3. Create tools (calculator, assessment)
4. Create infographics
5. Create case studies

**Outreach-Based:**
1. Guest posting (write for other HR blogs)
2. Expert mentions (get mentioned in articles)
3. Digital PR (press releases generate links)
4. Partnerships (co-marketing link exchanges)
5. Resource pages (get listed on authority sites)

**Strategy:**
- Target 20+ backlinks in Year 1
- Focus on quality over quantity
- Build relationships with HR bloggers
- Guest post on 12+ authority sites

## Technical SEO

### Site Architecture
- Clean URL structure (/blog/post-name, no dates)
- Logical hierarchy (category > subcategory > post)
- Internal linking structure (see link building above)
- XML sitemap (auto-generate + submit to GSC)

### Performance
- Mobile-first design (Google prioritizes)
- Page speed: Target < 2 seconds
- Core Web Vitals: All green
- Lazy loading images
- Minimize CSS/JavaScript

### Indexing
- Google Search Console: Monitor index status
- Sitemap: Submit and update regularly
- Robots.txt: Ensure not blocking indexable content
- Canonical tags: Avoid duplicate content issues

## Keyword Research Process

### Tools
- Google Keyword Planner (free)
- SEMrush (if budget allows)
- Ahrefs (if budget allows)
- AnswerThePublic (free, questions people ask)

### Process
1. Start with high-intent keywords
2. Analyze search volume vs competition
3. Evaluate intent (informational vs transactional)
4. Map keywords to content pieces
5. Prioritize by volume × relevance × achievability
